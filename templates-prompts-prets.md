# Templates de Prompts Prêts à l'Emploi

Cette collection de templates de prompts vous permet d'utiliser efficacement les assistants IA pour diverses tâches. Chaque template est optimisé pour maximiser la qualité des réponses.

## Rédaction et Communication

### Email Professionnel
```
Rédige un email professionnel pour [contexte/destinataire].

Objet : [sujet concis]

Contenu à couvrir :
- [point principal 1]
- [point principal 2]
- [appel à action]

Ton : [professionnel/amical/formel]
Longueur : [courte/moyenne/longue]
```

**Exemple concret :**
```
Rédige un email professionnel pour confirmer un rendez-vous client.

Objet : Confirmation de notre réunion du 15 mars

Contenu à couvrir :
- Rappeler l'objet de la réunion
- Confirmer l'heure et le lieu
- Fournir l'ordre du jour
- Demander confirmation

Ton : professionnel
Longueur : courte
```

### Rapport de Réunion
```
Crée un compte-rendu de réunion basé sur ces notes :

Participants : [liste]
Date/heure : [détails]
Objet : [sujet principal]

Points discutés :
- [point 1]
- [point 2]

Décisions prises :
- [décision 1]
- [décision 2]

Actions à suivre :
- [responsable] : [tâche] - [échéance]

Format : Structure claire avec sections
```

## Recherche et Analyse

### Analyse de Document
```
Analyse ce document : [insérer le texte ou décrire le document]

Identifie :
- Le sujet principal
- Les arguments clés
- Les conclusions
- Les points forts/faibles

Format : Structure avec sections claires
Niveau de détail : [sommaire/détaillé]
```

### Recherche Web
```
Effectue une recherche sur [sujet] et fournis-moi :

Informations clés :
- Définition et contexte
- Tendances actuelles
- Statistiques importantes
- Experts et sources fiables

Sources : Au moins 3 références récentes
Format : Points bullet clairs
```

### Synthèse d'Articles
```
Synthétise ces articles/sources sur [thème] :

Points communs :
- [insight partagé]

Différences :
- [divergence 1]

Conclusions globales :
- [synthèse principale]

Implications pratiques :
- [application concrète]
```

## Apprentissage et Formation

### Explication Simplifiée
```
Explique-moi [concept complexe] comme si j'avais 10 ans.

Utilise :
- Analogies concrètes
- Exemples quotidiens
- Pas de jargon technique

Vérifie ma compréhension avec une question à la fin.
```

### Plan d'Apprentissage
```
Crée un plan d'apprentissage pour maîtriser [compétence/sujet].

Niveau actuel : [débutant/intermédiaire/avancé]

Durée souhaitée : [mois]

Modules :
1. [module 1] : [objectifs] - [ressources]
2. [module 2] : [objectifs] - [ressources]

Évaluation : [comment mesurer les progrès]
```

## Techniques Avancées de Prompting

### Few-Shot Learning
```
Voici des exemples de [tâche] :

Exemple 1 :
Input : [entrée]
Output : [sortie souhaitée]

Exemple 2 :
Input : [entrée]
Output : [sortie souhaitée]

Maintenant, applique cela à : [nouvelle tâche]
```

### Chain of Thought
```
Résous ce problème étape par étape :

Problème : [description]

Étape 1 : Analyser la situation
Étape 2 : Identifier les variables clés
Étape 3 : Appliquer la logique appropriée
Étape 4 : Calculer le résultat
Étape 5 : Vérifier la cohérence

Conclusion : [résultat final]
```

### Role-Playing
```
Tu es un [rôle expert, ex: chef cuisinier, détective, coach sportif].

En tant que [rôle], aide-moi avec [problème].

Applique ta [expertise spécifique] pour :
- Analyser la situation
- Proposer des solutions
- Donner des conseils pratiques

Réponds comme si tu étais vraiment ce [rôle].
```

---

*Conseil : Personnalisez ces templates selon vos besoins spécifiques. Commencez par les plus simples et affinez progressivement vos prompts pour obtenir de meilleurs résultats.*
